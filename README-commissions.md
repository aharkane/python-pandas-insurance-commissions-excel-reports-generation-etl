# Insurance Broker Commission Analysis

Python-based ETL automation for processing multi-broker insurance commission data with automated Excel report generation.

## Overview

Automated pipeline that processes consolidated commission data from 7 insurance brokers, aggregates by business type (Health/Property & Casualty), and generates individualized Excel reports with organized worksheets.

## Key Features

- Processes commission data from 7 brokers across health and insurance sectors
- Handles positive commissions and negative adjustments (refunds)
- Automated Excel file generation with organized worksheets by business type
- Data aggregation and categorization by broker and business category

## Technology Stack

| Component | Technology |
|-----------|------------|
| **Language** | Python 3.x |
| **Data Processing** | Pandas, NumPy |
| **File I/O** | openpyxl/xlsxwriter |
| **Environment** | Jupyter Notebook |

## Results

- Automated commission aggregation and categorization
- Generated 7 individualized Excel reports
- Eliminated manual data segregation
- Streamlined financial reporting workflow
